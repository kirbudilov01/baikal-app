# Как посмотреть приложение на iPhone

## Вариант 1. Быстро, через Expo Go

Это самый быстрый вариант для теста интерфейса.

1. Установить **Expo Go** из App Store.
2. На Mac запустить:

```bash
npm run iphone
```

3. Отсканировать QR-код камерой iPhone или открыть URL из Expo.

Условия:
- iPhone и Mac должны быть в одной Wi-Fi сети;
- VPN/firewall может мешать локальному подключению.
- при первой заявке iPhone спросит доступ к камере, фото и геопозиции.

Если локальная сеть не работает, запустить tunnel:

```bash
npx expo start --tunnel
```

Что можно проверить в текущем демо:
- главный экран и процесс "Фото -> Точка -> Статус";
- съемку фото прямо с камеры;
- выбор фото из галереи;
- запрос текущей геопозиции;
- заполнение описания;
- автосохранение черновика заявки после выхода/перезапуска;
- очистку черновика;
- отправку заявки;
- экран успеха;
- появление заявки в списке;
- фильтры заявок;
- карту и профиль.

## Вариант 2. Поставить как отдельное приложение на iPhone

Это даст иконку приложения на домашнем экране, без Expo Go.

Нужно:
- Apple Developer Program аккаунт;
- Expo/EAS аккаунт;
- добавить iPhone как тестовое устройство во время EAS-сборки;
- собрать internal iOS build.

Команда:

```bash
npx eas-cli login
npx eas-cli build --platform ios --profile preview
```

После сборки EAS даст ссылку/QR. Открыть ее на iPhone и установить приложение.

Важно:
- без Apple Developer аккаунта нормальную `.ipa` для установки на физический iPhone собрать нельзя;
- для App Store/TestFlight используется production/TestFlight-процесс, это следующий этап после internal build.
